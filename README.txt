INSTRUKSI UMUM (WAJIB)
----------------------
1. Siswa mengedit file HTML yang sudah disediakan (boleh ubah struktur) tetapi wajib memuat semua elemen yang ditugaskan di POIN 3 lembar UKK.
2. Wajib menggunakan CSS internal (<style> ... </style>) — **jangan** pakai file .css terpisah.
3. WAJIB memberi pewarnaan: gunakan perubahan warna pada teks atau background (minimal 2 elemen yang berwarna).
4. Foto: ambil dari folder foto yang sudah disiapkan, gunakan file sesuai nomor absen (misal: 1.jpg untuk absen 1).
5. Nama file HTML boleh tetap index.html. Jangan zip; push langsung ke repo (tambahkan commit message sesuai format).
7. Dilarang mengunduh ZIP atau mem-paste template luar yang tidak dikerjakan sendiri. Salin template hanya dari file yang diberikan guru.
8. Push projek yang sudah tuntas ke Github masing masing untuk selanjutnya di deploy di netlifay dengan domain nama_siswa_ukk_pw_2025.netlify.app


VARIASI PER ABSEN (TAMBAHAN - setiap siswa harus menambahkan sesuai nomor absen)
-------------------------------------------------------------------------------

1.  Tambahkan daftar HOBI (minimal 3 item) di bawah CV.
2.  Tambahkan tabel PENGALAMAN ORGANISASI (tahun, peran, deskripsi singkat).
3.  Tambahkan kutipan motivasi (1 baris, italics) di bawah foto.
4.  Tambahkan daftar keahlian SOFTWARE (misal: Canva, Photoshop) dengan level (Dasar/Menengah/Mahir).
5.  Tambahkan riwayat PRESTASI (jika tidak ada tulis "Belum ada").
6.  Tambahkan paragraf "TENTANG SAYA" minimal 3 kalimat.
7.  Tambahkan NAVIGASI sederhana (Home / CV / Portofolio — anchor saja).
8.  Tambahkan tabel JADWAL HARIAN (Senin–Jumat, 2 kolom: jam & kegiatan).
9.  Tambahkan daftar CITA-CITA (minimal 2) dengan singkat alasan.
10. Tambahkan daftar SOFTSKILL (minimal 4) dengan ikon teks (emoji allowed).
11. Tambahkan PORTOFOLIO MINI: 2 item (judul + 1 kalimat deskripsi + link GitHub project).
12. Tambahkan daftar KESUKAAN (makanan, warna, hobi singkat).
13. Tambahkan TIMELINE PENDIDIKAN (sebagai list terurut).
14. Tambahkan tabel SPESIFIKASI KOMPUTER/LAPTOP (CPU, RAM, Storage).
15. Tambahkan paragraf "PENGALAMAN BELAJAR TKJ" minimal 4 kalimat.
16. Tambahkan review singkat tentang mata pelajaran favorit (2–3 kalimat).
17. Tambahkan daftar 3 WEBSITE EDUKASI FAVORIT dan alasan singkat.
18. Tambahkan bagian "RENCANA SETELAH LULUS" (3 item).
19. Tambahkan REPRESENTASI STRUKTUR FOLDER (kode teks/ASCII) proyek Anda.
20. Tambahkan bagian "FUN FACT" (1–2 kalimat unik tentang diri).
21. Tambahkan TUTORIAL MINI (5 langkah singkat) tentang topik TKJ yang dikuasai.
22. Tambahkan tabel "TARGET BELAJAR TKJ SEMESTER INI" (target + deadline).
23. Tambahkan daftar PLUGIN/EXTENSION BROWSER YANG DIGUNAKAN (3 item).
24. Tambahkan daftar 3 CHANNEL YOUTUBE EDUKASI FAVORIT + satu kalimat alasan.
25. Tambahkan paragraf "ALASAN MEMILIH JURUSAN TKJ" minimal 3 kalimat.

CATATAN PENTING UNTUK SISWA
---------------------------
- Gunakan foto yang sudah disediakan sesuai nomor absen. Jika foto tidak ditemukan, lapor ke guru.
- Beri warna pada minimal dua elemen (misal: header background + teks tombol).
- Pastikan tag penutup lengkap (</html>, </body>, dll).
- Jika menambah script JS kecil dibolehkan
- Nama domain di netlifay WAJIB nama_siswa_ukk_pw_2025.netlify.app
